Version 0.1 — Fondations conceptuelles

Date : 26 juin 2026

Ajouts

* Définition du cadre conceptuel GVH Dynamic.
* Introduction des observables fondamentales :
    * D(t) : dissonance dynamique ;
    * S(t) : cohérence dynamique locale ;
    * R_super : fraction temporelle de régime supercritique.
* Définition qualitative du régime supercritique.
* Présentation des domaines d’application envisagés.
* Identification des limites de la version 0.1.
* Définition de la feuille de route des développements futurs.

Statut

Cette version constitue le noyau fondateur du programme GVH Dynamic. Les définitions mathématiques opérationnelles, les procédures de normalisation, les validations numériques et les comparaisons avec les méthodes existantes seront introduites dans les versions suivantes.
