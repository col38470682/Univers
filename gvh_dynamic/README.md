# GVH Dynamic

## Version 0.1 — Noyau minimal

GVH Dynamic est un cadre méthodologique destiné à l'étude des transitions de régime dans les systèmes dynamiques.

Cette première version établit les fondations conceptuelles du programme.

## Observables fondamentales

- D(t) : Dissonance dynamique
- S(t) : Cohérence dynamique locale
- R_super : Fraction temporelle de régime supercritique

## Statut

Cette version est conceptuelle.

Les définitions mathématiques opérationnelles seront introduites dans les versions suivantes.
