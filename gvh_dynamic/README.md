GVH Dynamic

Cadre méthodologique pour l’étude des transitions de régime dans les systèmes dynamiques

GVH Dynamic est un programme de recherche consacré au développement d’un cadre méthodologique destiné à caractériser les changements de régime dans les systèmes dynamiques.

Le projet repose sur trois observables fondamentales :

* D(t) : dissonance dynamique ;
* S(t) : cohérence dynamique locale ;
* R_super : fraction temporelle de régime supercritique.

⸻

Version actuelle

Version : 0.1

Cette première version établit les fondations conceptuelles du cadre. Elle définit les observables fondamentales et leur interprétation qualitative, sans introduire encore leur définition mathématique opérationnelle.

Objectifs

Le programme GVH Dynamic poursuit plusieurs objectifs :

* définir des observables locales des régimes dynamiques ;
* caractériser les transitions critiques ;
* comparer différents systèmes non linéaires ;
* développer un cadre reproductible reposant sur des méthodes numériques ouvertes.
