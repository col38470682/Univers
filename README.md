# Univers

## Programme de recherche

**Univers** est un programme de recherche consacré au développement d'une description relationnelle de l'Univers à travers trois axes complémentaires :

- **GVH Dynamic**
- **Volume Partagé**
- **VP-GVH**

Le projet rassemble les développements théoriques, les modèles mathématiques, les simulations numériques et les confrontations aux observations.

---

# Axes de recherche

## 1. GVH Dynamic

Cadre méthodologique destiné à l'analyse multi-échelle des systèmes dynamiques non linéaires.

Principaux thèmes :

- Cohérence dynamique
- Régimes critiques
- Diagnostics multi-échelles
- Invariant B_H
- R_super
- Applications aux systèmes chaotiques
- Applications cosmologiques

---

## 2. Volume Partagé

Cadre cosmologique fondé sur une dynamique relationnelle du partage du volume.

Principaux thèmes :

- Dynamique du volume partagé
- Compartiments d'occupation
- Échanges relationnels
- Cosmologie émergente
- Comparaison avec le modèle ΛCDM

---

## 3. VP-GVH

Intégration des observables GVH au cadre du Volume Partagé.

Les développements actuels portent notamment sur :

- Occupation dynamique
- Invariants relationnels
- Conservation du volume
- Interprétation émergente de l'espace-temps
