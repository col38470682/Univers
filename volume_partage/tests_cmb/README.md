# Tests CMB – Archive méthodologique

## Objectif

Cette archive rassemble les expériences menées pour rechercher une corrélation entre le gradient thermique du fond diffus cosmologique (CMB) et différents catalogues cosmologiques.

## Jeux de données

- Pantheon+
- SDSS DR16 Quasars
- Carte Planck Commander-Ruler Nside 256

## Tests effectués

- Corrélation Pearson
- Corrélation Spearman
- Test de permutation
- Masque galactique
- Contrôle par quadrants
- Contrôle par redshift
- Contrôle directionnel
- Permutation par blocs spatiaux

## Résultats

Les signaux initiaux observés disparaissent après contrôle des biais de sélection des surveys.

Les analyses montrent que les corrélations proviennent principalement de la géométrie des catalogues et de leur stratégie d'observation, plutôt que d'un effet physique du gradient thermique du CMB.

## Conclusion

Cette campagne constitue une validation méthodologique importante du protocole d'analyse.

Aucune signature robuste reliant directement le gradient thermique du CMB au redshift n'a été mise en évidence dans les catalogues étudiés.
